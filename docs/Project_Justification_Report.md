# Project Justification Report: Elmenus Management System

## Project Information
- **Project Name**: Elmenus Management System
- **Team Members**: Anas Mohamed Gamal, Ziad Gawad
- **Group**: G6
- **Date**: April 20, 2026

## 1. Executive Summary
The Elmenus Management System is a C++-based restaurant management application that handles user registration, order processing, delivery driver management, and earnings tracking.

## 2. Why Incremental Model?
The system naturally breaks down into independent increments:
- **Increment 1**: User Management (Complete user registration system)
- **Increment 2**: Order Processing (Complete order management system)

## 3. Why Incremental is Best:
✅ Logical standalone modules
✅ Early working software delivery
✅ Simple integration strategy
✅ Clear progress tracking

## 4. Technology Stack
- Language: C++17
- Compiler: GCC/Clang
- Version Control: Git/GitHub
- Code Quality: SonarCloud
- CI/CD: GitHub Actions

## 5. Success Metrics
- Code Coverage: >80% ✅
- Code Duplication: <10% ✅
- Critical Issues: 0 ✅
- Maintainability Rating: A ✅

## 6. Conclusion
The Incremental Model is optimal for the Elmenus Management System as it allows early delivery of working software and reduces integration complexity.
